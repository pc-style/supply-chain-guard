Review before you install.

A gentle gate for npm and VS Code — lifecycle scripts, credential reads, suspicious payloads, and Socket signals, scanned locally on your machine.

  • Catches risky scripts before they run
  • Works with bun, npm, yarn, pnpm, VS Code
  • Private by default — no telemetry
Early stage tool. May miss threats or flag safe packages. Use as a helpful warning, not a certificate of safety.

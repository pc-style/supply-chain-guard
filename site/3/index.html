Sheet 01 · Overview

Review before you install.

Local inspection of npm packages and VS Code extensions — lifecycle behavior, credential access, network patterns, optional Socket enrichment.

  • Pre-install gate for package managers
  • Structured JSON / Markdown reports
  • No outbound telemetry by default
Note: Early-stage tooling — treat output as advisory, not certification.
